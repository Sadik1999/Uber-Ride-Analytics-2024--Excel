Ride-Sharing Performance Dashboard (Excel)

An interactive Ride-Sharing Analytics Dashboard built using Microsoft Excel to analyze booking performance, revenue distribution, cancellation behavior, and customer satisfaction. This project transforms raw ride-booking data into actionable insights to help understand operational efficiency and customer trends in a ride-sharing service.

📊 Project Objectives

The objective of this project is to:

Clean and prepare ride-booking data

Analyze operational metrics such as ride completion and cancellations

Identify revenue patterns and payment preferences

Evaluate service quality through customer and driver ratings

Build an interactive dashboard for quick business insights

📈 Dashboard KPIs

The dashboard highlights key performance indicators:

Total Bookings

Completion Rate

Cancellation Rate

Total Revenue

Average Customer Rating

These KPIs provide a quick overview of platform performance and service reliability.

📊 Visual Analysis

The dashboard includes the following visualizations:

📉 Bookings Trend Over Time

A line chart showing how booking demand changes over time.

🚗 Vehicle Type Performance

A comparison of booking volume across different vehicle categories.

💳 Revenue Distribution by Payment Method

A donut chart highlighting the most commonly used payment methods.

❌ Cancellation Analysis

Breakdown of cancellations by drivers vs customers and the reasons behind them.

⭐ Ratings Analysis

Comparison of customer ratings vs driver ratings across vehicle types to assess service quality.

🎛 Dashboard Interactivity

The dashboard includes interactive slicers that allow users to filter the data by:

Date

Vehicle Type

Payment Method

This enables dynamic exploration of the dataset.

🛠 Tools & Skills Used

Microsoft Excel

Power Query (Data Cleaning)

Pivot Tables

Excel Charts & Dashboard Design

Data Analysis & KPI Development

💡 Key Insights

Most bookings are successfully completed, indicating reliable platform performance.

A portion of rides are cancelled by either drivers or customers, highlighting areas for operational improvement.

Certain payment methods contribute a larger share of total revenue.

Some vehicle categories generate higher booking demand.

Overall ratings indicate generally positive customer satisfaction.
